# EduLearn - 现代课程学习平台

## 项目简介

EduLearn是一个现代化的课程学习网站首页，采用纯静态HTML、Tailwind CSS和Font Awesome构建，具有响应式设计，适配各种屏幕尺寸。

网站包含以下核心功能模块：

- 导航栏（支持移动端菜单）
- 英雄区（突出网站价值主张）
- 课程分类（编程开发、数据分析、设计创意、商业管理）
- 热门课程展示
- 结构化学习路径
- 学员评价
- 常见问题（交互式折叠面板）
- 行动召唤区域
- 页脚信息

## 部署说明

这是一个纯静态页面项目，无需构建命令。部署到Cloudflare Pages的步骤如下：

1. 登录Cloudflare Pages控制台
2. 连接您的GitHub仓库（WUYan720/wzy-site）
3. 选择构建设置：
   - 框架预设：`None`
   - 构建命令：留空（无需构建）
   - 输出目录：`/`（根目录）
4. 点击"部署站点"按钮
5. 部署完成后，Cloudflare Pages会提供一个唯一的域名用于访问您的网站

由于项目是纯静态的，部署过程非常简单，不需要任何构建工具或依赖管理。
